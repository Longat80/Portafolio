Sobre mí

Soy un apasionado del desarrollo web Front End, especializado en crear interfaces modernas y responsivas con HTML, CSS y JavaScript. Me encanta aprender nuevas tecnologías para construir experiencias de usuario intuitivas y eficientes. Utilizo JSON para el manejo e intercambio de datos, y aplico SASS para optimizar y mantener estilos de forma escalable. Me apoyo en Bootstrap para acelerar el desarrollo y asegurar compatibilidad entre dispositivos, y en React para crear aplicaciones interactivas y modulares con un enfoque en rendimiento y mantenibilidad. Además, tengo experiencia utilizando Git como sistema de control de versiones, lo que me permite colaborar de forma ordenada y profesional en equipos de desarrollo.
